# Ansible #1
Плейбук использует следующие переменные:

| Переменная        | Описание                             |
|-------------------|--------------------------------------|
| `user: string`    | Имя нового пользователя              |
| `group: string`   | Имя новой группы                     |
| `pub_key: string` | Публичный SSH-ключ для входа по user |
